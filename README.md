# Who Wants to Be a Millionaire? (iOS App)(Команда №4)

Добро пожаловать в репозиторий нашего iOS-приложения "Кто хочет стать миллионером?"! Это проект, разработанный нашей командой как групповое задание. Приложение представляет собой интерактивную викторину, где игроки отвечают на вопросы, чтобы выиграть виртуальный миллион долларов.

## 📱 Описание

Наше приложение воссоздает атмосферу знаменитой телевикторины. Оно включает в себя:
*   **15 уровней**: Вопросы различной сложности (легкие, средние, трудные).
*   **Система подсказок**: "50/50", "Помощь зала", "Звонок другу".
*   **Таймер**: Ограниченное время (30 секунд) на обдумывание каждого вопроса.
*   **Денежное дерево**: Визуальное представление прогресса и несгораемых сумм ($5,000, $25,000 и $1,000,000).
*   **Звуковое сопровождение**: Уникальные звуки для различных игровых событий.
*   **Рекорды**: Сохранение лучшего результата пользователя.

## 🚀 Особенности

*   **Работа с сетью**: Интеграция с внешним API [Open Trivia Database](https://opentdb.com/) для динамической загрузки вопросов различной сложности.
*   **Кастомные UI компоненты**:
    *   `ExtendedHitButton`: Кнопка с увеличенной областью нажатия для улучшения UX.
    *   `AudienceHint`: Кастомный `UIView` для отображения анимированной гистограммы результатов подсказки "Помощь зала".
    *   `AnswerCell`: Кастомная ячейка для отображения денежного дерева.
*   **Логика**:
    *   Сложная логика работы подсказок, включая вероятностное распределение ответов для "Звонка другу" и "Помощи зала".
    *   Централизованное управление звуком (`SoundManager`).
    *   Централизованный менеджер состояния игры (`GameBrain`).
    *   Сохранение рекордов с помощью `UserDefaults`.
*   **Анимации**: Использование `UIView.animate` для различных эффектов (пульсация, мигание, рост столбцов).

## 🛠 Технологии

*   **Язык программирования**: Swift
*   **Фреймворки**: UIKit, AVFoundation
*   **Инструменты**: Xcode
*   **Работа с сетью**: URLSession
*   **Хранение данных**: UserDefaults
